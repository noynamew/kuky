Задание 1: Инициализация репозитория и базовые команды Git

Инструкции

  1. Установите Git на свой компьютер, если он ещё не установлен.
  2. Создайте папку для проекта и инициализируйте в ней Git-репозиторий командой git init.
  3. Создайте несколько текстовых файлов и добавьте их в репозиторий с помощью команды git add.
  4. Закоммитьте изменения с осмысленным сообщением, используя git commit -m "Ваше сообщение".
  5. Проверьте статус репозитория с помощью команды git status и историю коммитов с git log.

Цель

Освоить базовые команды Git для создания и отслеживания изменений в локальном репозитории.
